# 🍽️ Responsive Food Landing Page

A modern, mobile-first landing page for a food service business. Built with **HTML**, **CSS**, and **JavaScript**, the website features smooth animations, responsive design for all devices, and interactive elements like a toggleable mobile menu and testimonial slider.

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
  - Flexbox & Grid
  - Media Queries
  - Animations & Transitions
- **Vanilla JavaScript**
- **Google Fonts**: Poppins, Barlow Condensed, Roboto
- **Remix Icon**: For scalable icons

---

## 🔧 Features

✅ Fully responsive layout (mobile, tablet, desktop)  
✅ Navigation menu with mobile toggle button  
✅ Hero section with CTA  
✅ Featured dishes with star ratings  
✅ Explore and Chef introduction section  
✅ Testimonial slider with navigation bullets  
✅ Footer with newsletter subscription
